# **L**yric **A**nd **M**usic **B**eautiful **D**isplay **A**ggregator

*A light lrc-format lyric / spectrum visualizer*

LAMBDA is a terminal lyric and spectrum visualizer written to remind us of the times when the world gazed into the future through the lenses of computing and electronic music.

## Features
- Supports parsing of .lrc files
- Useable only on Linux distributions (for now)
- If no .lrc is provide or if explicitely selected, LAMBDA will act as a spectrum visualizer
- Displays nice visuals :D

## NOTE
This program displays .lrc lyric files that you _(the user)_ provide. **It does not** download, fetch, or distribute lyrics. You are solely responsible for ensuring you have the right to use any lyric files you load.
